Select a pattern to explore details, UML diagrams, and code examples:

Singleton

Ensure a class has only one instance and provide a global point of access.

Template Method

Defines the skeleton of an algorithm in the superclass but lets subclasses override specific steps.

Factory Method

Define an interface for creating an object, but let subclasses decide which class to instantiate.

Adapter

Convert the interface of a class into another interface clients expect. The Adapter pattern lets classes work together even if their interfaces are incompatible.
